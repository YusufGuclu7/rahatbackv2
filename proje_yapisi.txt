TOPLAM İSTATİSTİKLER
=====================
- Backend dosya sayısı: ~100 dosya (node_modules hariç)
- Frontend dosya sayısı: ~97 dosya (node_modules hariç)
- Toplam: ~197 dosya (node_modules ve backups hariç)

KULLANILAN TEKNOLOJİLER
========================
Backend:
- Node.js + Express.js
- Prisma ORM
- PostgreSQL/MySQL/MongoDB desteği
- JWT Authentication
- Passport.js
- Node-cron (Zamanlanmış yedekler)
- AWS S3 & Google Drive entegrasyonu
- Email servisi

Frontend:
- React.js
- Material-UI (MUI)
- AG-Grid
- Axios
- SweetAlert2
- React Router

ÖZEL ÖZELLIKLER
================
✓ Otomatik veritabanı yedekleme sistemi
✓ Çoklu veritabanı desteği (PostgreSQL, MySQL, MongoDB)
✓ Cloud storage entegrasyonu (S3, Google Drive)
✓ Zamanlanmış yedekler (cron jobs)
✓ Email bildirimleri
✓ Kullanıcı yönetimi ve rol tabanlı yetkilendirme
✓ Admin paneli
